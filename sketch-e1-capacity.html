What if Pattern is a "how" not a "what"?
Key and Tuning define what you're playing. Pattern defines how you see it on the fretboard. This puts it in the pill row with the other display/playback options. 3 pill groups fits, but it's the practical max.
The "What" vs "How" Framework
What = Selector Row (E1 underlines)
Things that change what music is shown. They open sheets or cycle through lists. The value is a noun.

Chords: Key (C Major), Voicing (All/Triads/Shells/Full), Tuning (Standard/Drop D/...)
Scales: Key (A Minor), Tuning (Standard/...)
Maybe: Pattern? Capo?

How = Pill Row (segmented toggles)
Things that change how it's displayed or played. Binary or small-set choices. Quick tap, no sheet needed.

Chords: Notes/Intervals, Strum/Block
Scales: Notes/Intervals, Asc/Desc
Maybe: Pattern (Boxes/3NPS/Full)?

Capacity Summary
Selector row: 3 comfortable, 4 tight, 5 needs abbreviations
Pill row: 2x2 comfortable, 3 groups fits, 3+3+2 is the max
